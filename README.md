mcli
=====

A Minecraft mod manager made for command line interface.

Installation Presquiste
-----------------------

- Modded Minecraft of supported modloader.

- Installed Python.

- Dependancies: `pip install colorama requests`.

Installation
------------

1. Copy or clone this repository.

2. Create alias to the repository, something of `alias mcli "python /path/to/repo"`.

3. [Usage](#usage)

Usage
-----

Initiate cli with `mcli` or your's alias. For help enter `mcli help`.
